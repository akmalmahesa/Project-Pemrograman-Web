# Website Penyewaan Kendaraan (Rental Mobil/Motor/Sepeda)

## Deskripsi Proyek
Website ini dibuat untuk memudahkan pengguna dalam **menyewa kendaraan** seperti mobil, motor, atau sepeda secara **harian atau mingguan**.  
Proyek ini dikembangkan secara **kelompok** dengan pembagian peran yang terstruktur, mencakup frontend, backend, database, dan pengujian.

Website ini memungkinkan pengguna untuk:
- Melihat daftar kendaraan yang tersedia  
- Melakukan pemesanan (booking) kendaraan  
- Melihat status kendaraan (tersedia/disewa)  
- Login sebagai pengguna atau admin  

---

## Fitur Utama
1. **Login Pengguna & Admin**  
   - Sistem autentikasi untuk pengguna dan admin.  
   - Admin dapat menambah, mengedit, dan menghapus data kendaraan.  

2. **Daftar Kendaraan yang Tersedia**  
   - Menampilkan daftar kendaraan lengkap dengan detail seperti nama, jenis, harga sewa, dan status.

3. **Form Booking Sewa**  
   - Pengguna dapat mengisi formulir untuk melakukan penyewaan kendaraan berdasarkan tanggal dan durasi.

4. **Status Kendaraan (Tersedia/Disewa)**  
   - Setiap kendaraan memiliki status yang diperbarui secara otomatis berdasarkan transaksi.
